## 📖 简介

这是一个个人技术笔记仓库，记录了在软件开发、工具使用、网络配置等方面的实践经验和技术总结。内容主要为中文，部分技术文档为英文。

## 📂 目录结构

- **AI/** - AI 工具与开发环境（Claude、Cursor、Gemini 等）
- **apple/** - macOS 软件推荐、使用技巧和配置
- **programming/** - 编程工具和编辑器（Zed、VSCode、Bun、pnpm 等）
- **python/** - Python 相关工具、类型注解、远程调试等
- **science/** - 网络代理工具和配置（Shadowrocket、Sing-box、Egern 等）
- **net_utils/** - 网络工具（EasyTier、Tailscale Funnel 等）
- **software/** - 软件推荐和配置
- **tampermonkey/** - 浏览器用户脚本
- **assets/** - 图片和其他资源文件
