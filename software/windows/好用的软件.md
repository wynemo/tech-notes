+ WinDirStat

WinDirStat 是一个适用于各种版本的 Microsoft Windows 的磁盘使用情况统计查看器和清理工具 https://windirstat.net/

- 可视化显示磁盘空间使用情况，使用彩色方块图直观展示文件大小分布
- 支持扫描整个硬盘或指定文件夹，快速找出占用空间最大的文件和文件夹
- 特别适合清理磁盘空间，找出那些被遗忘的大文件
